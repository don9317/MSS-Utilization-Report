# MSS Utilization Dashboard v1.0 Release Notes

Initial clean release.

Included fixes and rules:
- Correct day-of-week by parsing reservation dates as local dates.
- Multi-select court/space dropdown that closes after selection.
- Space Setup panel for changing facility hours and parent/child rules.
- Basketball parent/child rules: Courts 1–4 and divided courts a/b/c.
- Baseball parent/child rules: Full Field, Tunnels 1–7, Practice Turf.
- Adjusted rentable capacity denominator to avoid double-counting full courts and subdivided courts.
- Embedded TM sample data from `TM MSS 1_1 to 4_30.csv`.

Recommended next version:
- Save named facility configurations.
- Add one-click facility presets.
- Add export-to-PDF or snapshot report.
